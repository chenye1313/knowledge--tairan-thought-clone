# 核心术语表 v1

> 说明：本表先统一口径，后续需要继续补来源、例句、反例和边界声明。

| 术语 | 初步定义 | 所属层级 | 边界提醒 |
|---|---|---|---|
| 人生游戏 | 把人生看作体验、成长、觉醒和创造的过程 | 世界观 | 不是轻视痛苦，而是改变解释框架 |
| 功课 | 外在事件触发内在成长的入口 | 方法论 | 不是忍耐、自责或合理化伤害 |
| 向内求 | 从外部归因转向内在状态、定义、关系和功课的觉察 | 方法论 | 不等于否认外部责任 |
| 明心见性 | 看见情绪、故事、执着背后的真实内在结构 | 三大功课 | 需要落到具体事件和行为改变 |
| 神游八荒 | 借助更大时间、空间、人物、自然和智慧视角破除限制 | 三大功课 | 不是逃避现实的幻想 |
| 逍遥有为 | 在当下敞开自己，依灵感创造，不被结果绑架 | 三大功课 | 不是消极无为，也不是强行成功学 |
| 抽离 | 从故事、情绪和身份中退出来观察 | 技法 | 不是冷漠或压抑情绪 |
| 填空 | 在事件中看见自己的匮乏、依赖、控制、恐惧、贪婪等 | 技法 | 不能变成道德审判 |
| 感恩 | 将对抗、抱怨、受害者状态转为接纳和能量回收 | 技法 | 不等于纵容不合理关系 |
| 重塑 | 重新定义事件、关系、身份和未来行动 | 技法 | 必须连接行动，否则停留在口号 |
| 能量 | 泰然体系中描述状态、关系、感受和承载力的核心语言 | 世界观/方法论 | 暂不作为科学物理概念处理 |
| 频率 | 人的状态、念头、情绪和外在吸引之间的同频语言 | 世界观/方法论 | 避免直接证明式因果断言 |
| 定义 | 人对事件、自己、他人、财富、家庭的解释框架 | 方法论 | 定义改变需要行为验证 |
| 天命/使命 | 个体在生命中要承担和创造的方向 | 个人成长 | 需避免神秘化和过度承诺 |
| 父母就是风水 | 父母、根、祖先和家庭关系对人生命状态的根本影响 | 家庭家族 | 不能简化为孝顺万能论 |
| 家风 | 一个家族长期形成的价值、秩序和行为模式 | 家庭家族 | 需要区分口号和真实行为 |
| 家训 | 家族可传承的原则、规则和教化文本 | 家庭家族 | 不能写成空泛标语 |
| 家规 | 家族成员共同遵守的具体约束 | 家庭家族 | 需要可执行、可讨论、可迭代 |
| 家魂 | 家族精神、使命、核心气质和共同愿景 | 家庭家族 | 需从真实故事中提炼 |
| 家族理事会 | 家族重大事务、教育、创业、矛盾协调的协商机制 | 家庭家族 | 需设计治理边界，防止家长制 |
| 环境科学 | 对风水的现代化表达，关注人与空间、家庭、事业状态的互动 | 环境 | 需要边界声明，不做科学过度包装 |
| 数字能量 | 对手机号、数字组合和状态之间关系的体系内解释 | 号码 | 只能作为辅助工具，不能决定论 |
