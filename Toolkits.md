## 附录A 三大功课一页练习表

### 使用说明

这份表不是为了把生命变成表格，而是在情绪最乱、事情最急、头脑最容易掉回旧剧情时，给自己一个可落脚的顺序。

```text
先明心见性，看见卡点。
再神游八荒，打开视角。
最后逍遥有为，落到行动。
```

不要一开始就问“我该怎么办”。很多时候，人之所以不知道怎么办，是因为还没看清自己卡在哪里。也不要只停在“我看见了”。看见之后没有新的定义、新的选择、新的行动，功课还没有落地。

### A1. 明心见性

```text
事件：
涉及的人：
发生的时间：
我当时最强的情绪：
我脑子里自动编出的故事：
我把自己放在什么角色：
  受害者 / 拯救者 / 控制者 / 讨好者 / 证明者 / 被抛弃的人 / 必须负责所有人的人
如果只看事实，不加解释，这件事可以怎么描述：
这件事照见我对____的依赖：
这件事照见我对____的控制：
这件事照见我在____上的匮乏：
这件事照见我一直想证明____：
这件事照见我害怕____：
这件事照见我不愿面对____：
```

边界提醒：填空不是自责。如果一段关系里有暴力、欺骗、严重伤害，填空不是说“都是我吸引来的，所以我活该”。更准确的口径是：外在责任归外在责任，我的功课归我的功课，边界归边界，行动归行动。

### A2. 抽离

```text
如果我是旁观者，我会看到什么：
如果这件事发生在朋友身上，我会怎么提醒他：
如果十年后的我看今天，他会怎么理解：
如果这是人生游戏里的一个关卡，这一关叫什么：
我现在有情绪，但我不等于这个情绪。我可以站到的新位置是：
```

抽离不是冷漠，也不是压抑情绪。抽离只是从“我就是这个情绪”里退一步。有空间，才有选择。

### A3. 感恩

感恩不是替坏事洗白。感恩的意思是：我不再让这件事只负责伤害我，我要从里面拿回成长材料。

```text
这件事提醒我：
它让我看见的旧模式：
它逼我面对的事实：
它提醒我需要建立的边界：
它让我长出的能力：
它让我重新理解的关系：
我感谢自己没有继续沉睡在：
```

有些人暂时做不到感谢对方，这是正常的。可以先从感谢自己愿意面对开始。真实比漂亮更重要。


### A8. 使用边界

功课不是万能解释，不是压抑情绪，不是替伤害洗白，不是表演给别人看。严重抑郁、创伤、暴力关系、法律纠纷、重大财务风险、医疗问题和人身安全问题，应优先寻求专业支持。向内求不是孤立无援，而是在拿回主导权之后，更清醒地选择需要的外部帮助。

## 附录C 环境观察清单

### C1. 使用原则

```text
不恐吓。
不决定论。
先观察，再调整，再验证。
```

不要一进门就给自己判刑。更稳的方式是问：这个空间哪里让我紧，哪里让我乱，哪里让我失去托底感，哪里让我难以聚焦，哪里可以通过整理、光线、动线、边界来改善。

### C2. 基础状态

```text
房间是否长期杂乱：
入口是否堵塞：
光线是否昏暗：
空气是否浑浊：
是否有长期不用却占空间的物品：
是否有坏了却一直不修的东西：
是否有让人一看就心烦的角落：
是否有明显潮湿、异味、漏水：
```

调整顺序：

```text
先清垃圾。
再断舍离。
再修坏物。
再通风采光。
再谈布局。
```

### C3. 靠山

观察：

```text
床头是否有稳定靠背：
书桌或办公位背后是否太空：
沙发背后是否有墙或稳定承托：
家里北方位置是否过于空、乱、破、漏：
主要负责人是否长期坐在背后被人走动的位置：
```

调整：

```text
让床头有靠。
让办公位背后稳定。
让沙发不悬空。
北方位置保持整洁、稳定、有承载感。
必要时用柜、书架、稳重物件增加托底感。
```

边界提醒：靠山不只是物理空间，也包括能力、信用、家庭支持、贵人关系和内在定力。

### C4. 明堂

观察：

```text
入户后是否一眼压抑：
客厅前方是否太堵：
窗前是否堆满杂物：
办公桌前方是否没有展开空间：
南方位置是否受压、阴暗、杂乱：
孩子学习区前方是否过于逼仄：
```

调整：

```text
清出前方视野。
让光进入空间。
减少面对面的压迫物。
把前方最乱的角落整理出来。
在适当位置增加开阔、明亮、向上的象。
```

### C5. 青龙与白虎

观察：

```text
房子东侧或东南侧是否明显低弱、破败、杂乱：
西侧或西南侧是否过高、过压、过尖、过动：
家里是否长期口舌多、争执多：
女性是否长期紧绷或情绪压力大：
孩子学习区是否缺少生发支持：
```

调整思路：

```text
生发侧太弱：增加整洁、书籍、绿植、学习和生发之象。
压力侧太强：减少尖锐、杂乱、压迫和过度动线。
两侧都乱：先清理，再谈象。
```

边界提醒：家庭争吵不能只归因给空间，还要看沟通方式、夫妻关系、亲子秩序、财务压力和个人功课。

### C6. 卫生间与排污系统

观察：

```text
是否长期有异味：
是否潮湿发霉：
门是否长期敞开对着重要位置：
马桶盖是否长期打开：
排水是否不畅：
镜子是否污浊：
是否堆放过多杂物：
是否缺少通风和光线：
```

调整：

```text
保持通风。
保持干燥。
及时维修漏水和堵塞。
减少杂物。
不用时关门、盖盖。
让清理系统真的清理，而不是制造新的浊气。
```

### C7. 厨房与火

观察：

```text
厨房是否油污重：
炉灶是否长期不用或坏损：
水火是否明显冲突：
西北方是否厨房火气过重：
做饭的人是否长期带着怨气：
家里是否很少一起吃饭：
```

调整：

```text
保持灶台清洁。
让厨房有烟火气，但不要脏乱。
减少水火直接冲突。
西北火重时，谨慎使用转化思路，不要简单用水压火。
把吃饭重新变成家庭连接，而不是只解决饥饿。
```

### C8. 外部环境

观察：

```text
门窗是否正对强烈路冲：
是否有尖角、壁刀、天斩等压迫感：
是否长期面对坑、破败、垃圾场：
是否噪音严重：
是否缺少采光和通风：
是否出入动线让人烦躁：
```

处理原则：

```text
先判断实际影响：光、风、噪音、安全、隐私、动线。
再判断心理感受：压迫、焦虑、不安、分散。
最后才考虑象的调整。
```

### C9. 调整验证

```text
调整前，我的状态是什么：
调整了什么：
调整后7天，身体感受有什么变化：
调整后30天，家庭互动有什么变化：
调整后90天，工作效率、睡眠、情绪有什么变化：
有没有可能是其他因素导致：
哪些调整有效：
哪些调整无效：
下一步是否需要专业支持：
```

号码服务需求，而不是人被号码绑架。

```text
它用于帮助学习者观察状态、角色、阶段需求与可能缺口。
它不构成命运判断、医疗判断、法律判断、投资判断、婚姻裁决或任何确定性承诺。
任何号码分析都必须结合个人现实情况，并回到三大功课、家庭关系、事业行动、环境整理和专业支持中验证。
```

### E2. 不说的话

```text
你这个号码一定破财。
你这个号码一定离婚。
你这个号码一定有灾。
你这个号码一定没有贵人。
你只要换号码就能翻身。
这个号害你一辈子。
不换就完了。
```

### E3. 更稳的说法

```text
这个号码在体系内呈现出某种倾向。
它可能与你当前状态有相似之处。
我们可以结合你的性别、年龄、职业、婚姻、使用时间和现实处境一起看。
如果你正好有相关问题，可以把它当作一个提醒。
真正改变，还要回到功课、关系、环境和行动。
```

### E4. 咨询者伦理

```text
不恐吓。
不绝对化。
不制造依赖。
不暗示只有自己能救对方。
不把复杂人生简化成一个号码。
不替代医疗、法律、心理、财务等专业建议。
不在对方脆弱时放大焦虑。
不承诺换号后必然发生某种结果。
不传播对方手机号和隐私。
```

### E5. 学习者自检

```text
我是在学习观察，还是追求控制？
我是在帮助别人清醒，还是证明自己很准？
我是在提醒别人行动，还是让别人依赖我？
我有没有因为一个组合，就忽略了这个人的真实人生？
我有没有把恐惧包装成专业？
我有没有把辅助工具放到根本位置？
```

## 附录F 核心术语索引

### 世界观类

**人生游戏**：把人生看作体验、成长、觉醒和创造的过程。边界：不是轻视痛苦，而是改变解释框架。

**能量**：泰然体系中描述状态、关系、承载力、感受和行动动力的核心语言。边界：不是物理学概念，不用伪科学方式证明。

**频率**：人的状态、念头、情绪和外在关系之间相互呼应的表达方式。边界：不能写成绝对因果，不能说“频率低，所以一切坏事都是你吸引的”。

**定义**：人对事件、自己、他人、财富、家庭和未来的解释框架。边界：定义改变必须通过行为验证。

### 方法论类

**功课**：外在事件触发内在成长的入口。边界：不是忍耐、自责或合理化伤害。

**向内求**：从外部归因转向内在状态、定义、关系和功课的觉察。边界：不等于否认外部问题。

**明心见性**：看见情绪、故事和执着背后的真实内在结构。边界：不能停在理解自己，还要连接行为改变。

**神游八荒**：借助更大的时间、空间、人物、自然和智慧视角，破除当下限制。边界：不是逃避现实的幻想。

**逍遥有为**：在当下敞开自己，依灵感创造，不被结果绑架。边界：不是躺平，也不是焦虑成功学。

**抽离**：从故事、情绪和身份中退出来观察。边界：不是冷漠，也不是压抑情绪。

**填空**：在事件中看见自己的匮乏、依赖、控制、恐惧、证明等缺口。边界：不能变成道德审判，也不能把所有问题都怪自己。

**感恩**：将对抗、抱怨、受害者状态转为接纳、看见和能量回收。边界：不等于纵容不合理关系，不等于替坏事洗白。

**重塑**：重新定义事件、关系、身份和未来行动。边界：必须连接行动，否则停留在口号。

### 家庭家族类

**父母就是风水**：父母、根、祖先和家庭关系对人生命状态有根本影响。边界：不能简化成孝顺万能论，也不能要求受伤者无边界靠近伤害源。

**家风**：一个家族长期形成的价值、秩序和行为模式。边界：不是墙上的口号，要看日常行为。

**家训**：家族可传承的原则、规则和教化文本。边界：不能空泛，要能进入具体生活。

**家规**：家族成员共同遵守的具体约束。边界：需要可执行、可讨论、可迭代。

**家魂**：家族精神、使命、核心气质和共同愿景。边界：不能编造，要从真实故事里长出来。

**家族理事会**：家族重大事务、教育、创业、矛盾协调的协商机制。边界：防止变成家长制，也不能替代法律和专业咨询。

### 环境与数字类

**环境科学**：对风水的现代化表达，关注人与空间、家庭、事业状态的互动。边界：不做科学过度包装，不恐吓，不决定论。

**靠山**：空间和人生中的托底感、承载感、支持感。边界：不只是物理空间，也包括能力、信用、家庭和贵人。

**明堂**：前方的开阔、光亮、未来感和展开空间。边界：不是豪宅概念，小空间也可以通过整理和光线形成明堂。

**青龙白虎**：环境中左右力量、生发与压力、助力与边界的观察语言。边界：不机械套公式，不把家庭矛盾单独归因给空间。

**卫生间**：家庭排污、清理、隐蔽系统。边界：不妖魔化卫生间，重点是通风、明亮、干净、干燥和位置管理。

**火烧天门**：体系内对厨房位于西北乾位、火金关系失衡的环境说法。边界：只作环境观察，不作健康诊断。

**数字能量**：对手机号、数字组合和状态之间关系的体系内解释。边界：只能辅助，不能决定论，不能替代行动和专业判断。

**手机号七步曲**：基本信息、分阴阳、排能量、辨大势、看重点、找缺陷、抓需求的观察流程。边界：流程用于防止粗糙判断，不用于给人定命。
